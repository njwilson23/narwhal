Narwhal Toolkit for Oceanography

Narwhal is an experimental Python library built on pandas and matplotlib for manipulating and visualizing oceanographic survey data.

Cast and CastCollection classes organize data in a way that is easy to manipulate and build analyses upon. For example, the Cast class has built-in methods for (incomplete list)

  • interpolation
  • density and depth calculation
  • buoyancy frequency calculation
  • baroclinic mode analysis
  • linear water mass analysis
More, specialized techniques can be added by subclassing the generic Cast object.

Cast and CastCollection data can also serialize itself as zipped JSON objects for data storage and sharing. WOCE CTD casts in NetCDF format may be read directly.

The narwhal.plotting submodule contains convenience methods for creating attractive T-S diagrams, cast plots, and section plots. The data to the right and below is from the WOCE P17N line, collected on a cruise by the Thomas G. Thomson.

Narwhal provides a wrapper for the Gibbs Seawater Toolbox in the narwhal.gsw submodule, making things like the following possible:

density = narwhal.gsw.rho(cast["abssal"], cast["constemp"], cast["pres"])

INSTALLATION

git clone https://github.com/njwilson23/narwhal.git
pip install -r narwhal/requirements.txt
pip install narwhal

DEPENDENCIES

  • Python 2.7+ or Python 3.3+
  • pandas
  • numpy
  • scipy
  • karta
  • requests
  • dateutil
  • six
  • matplotlib (optional; for plotting)
  • C-compiler (optional; for GSW)

TESTING

To run all unit tests, execute

python tests/runtests.py

Narwhal is experimental. Also consider python-oceans and oce (R).
